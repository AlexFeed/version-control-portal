# VCP Frontend (мокап без бэкенда)

Фронтенд-макет Version Control Portal для практики. Данные — моковые, хранятся в памяти приложения (сбрасываются при перезагрузке страницы), бэкенда нет.

## Запуск

```bash
npm install
npm run dev
```

Откройте адрес, который выведет Vite (обычно http://localhost:5173).

## Вход

На экране входа email и пароль не проверяются — выберите роль (Viewer/Developer/Admin) в выпадающем списке, чтобы посмотреть, как меняется доступ к действиям.

## Тесты

```bash
npm run test
```

Автотестами покрыт слой моковых API (`src/api/*.test.ts`) — создание/изменение/удаление проектов, версий и пользователей. Остальное (страницы) проверялось вручную в браузере.
